## General info

This project is primarily developed using the ["https://rickandmortyapi.com"](https://rickandmortyapi.com) API and provides rich information about the characters and episodes of the Rick and Morty animated series. The project has been made in a design compatible with iPhone, iPad and Mac devices. In the project, UIKit was used in general, SwiftUI was used only in the settings page.
	
## Technologies and 3rd party libraries
Project is created with:
* UIKit
* SwiftUI
* MVVM-Cordinator Design Pattern
* Moya
* RxSwift
* RxCocoa
* RxDataSource
* MapKit
* SafariServices
* StoreKit
* SDWebImage
